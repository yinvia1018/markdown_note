# 使用方法:
## ①在 docs 文件夹 .vuepress下 ，在 .vuepress文件夹下的config.js配置.md文件的路径 
## ②在 docs 文件夹下添加需要展示的.md文件



## 运行项目

  ```bash
  yarn docs:dev
  ```
## 访问
```
  http://localhost:8080/doc/
```
